### setup

`yarn`

### run zksync local chain

```shell
git clone https://github.com/matter-labs/local-setup.git
cd local-setup
./start.sh
```

### deploy

create .env file and add the line `NODE_ENV="test"`.

then run:

```shell
yarn hardhat compile
yarn hardhat deploy-zksync --script deploy/deploy.ts
```
